# server.properties

修改服务器的运行配置，以此来定制一些基本功能。

## `server-ip`
| Data Type | Default Value |
|-----------|---------------|
| String    | 0.0.0.0       |

服务器IP地址

## `enable-query`
| Data Type | Default Value |
|-----------|---------------|
| Boolean   | on            |

是否允许通过 Query 查询服务器信息

详见：https://wiki.vg/Query
