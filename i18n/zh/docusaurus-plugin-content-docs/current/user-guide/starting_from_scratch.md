# 从0开始的 Nukkit 服务器搭建教程

欢迎来到 Nukkit 服务器搭建教程!本教程将引导你从零开始搭建一个 **Nukkit 服务器**，让你能够运行自己的 **Minecraft 基岩版服务器**。

## 教程内容

本教程已拆分为多个章节，方便你按需查阅:

### 📖 章节导航

1. **[准备工作](getting-started/preparation.md)**
   了解系统要求，安装 Java 运行环境

2. **[下载和安装](getting-started/installation.md)**
   下载 Nukkit-MOT，创建服务器文件夹和启动脚本

3. **[配置服务器](getting-started/configuration.md)**
   自定义服务器设置，调整游戏参数

4. **[安装插件](getting-started/plugins.md)**
   扩展服务器功能，安装和管理插件

5. **[连接服务器](getting-started/connect.md)**
   让玩家加入你的服务器

6. **[常见问题](getting-started/troubleshooting.md)**
   解决搭建过程中的常见问题

## 快速开始

如果你想快速体验，可以遵循以下最简步骤:

1. **安装 Java 17+** - [准备工作](getting-started/preparation.md)
2. **下载 Nukkit-MOT** - [下载链接](https://motci.cn/job/Nukkit-MOT/job/master/lastSuccessfulBuild/artifact/target/Nukkit-MOT-SNAPSHOT.jar)
3. **创建启动脚本** - 参考 [安装教程](getting-started/installation.md)
4. **启动服务器** - 双击运行脚本
5. **连接游戏** - 使用 `127.0.0.1:19132`
