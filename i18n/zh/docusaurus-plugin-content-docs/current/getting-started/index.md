# 从0开始的 Nukkit 服务器搭建教程

欢迎来到 Nukkit 服务器搭建教程!本教程将带你从零开始,一步步搭建属于自己的 Minecraft 基岩版服务器。

## 教程简介

本教程适合:
- 没有服务器搭建经验的新手
- 想要快速上手 Nukkit 的开发者
- 需要参考文档的服务器管理员

你将学会:
- 搭建完整的 Nukkit 服务器
- 配置服务器参数
- 安装和管理插件
- 解决常见问题

## 教程章节

按照以下顺序阅读,可以完整掌握 Nukkit 服务器的搭建:

### 1. [准备工作](preparation.md)
在开始之前,你需要准备好必要的环境:
- 检查系统要求
- 安装 Java 运行环境
- 验证安装是否成功

### 2. [下载和安装](installation.md)
获取 Nukkit 并完成基本设置:
- 下载 Nukkit-MOT
- 创建服务器文件夹
- 编写启动脚本
- 首次启动服务器

### 3. [配置服务器](configuration.md)
自定义你的服务器设置:
- 编辑 server.properties
- 调整游戏模式和难度
- 设置最大玩家数
- 配置白名单和权限

### 4. [安装插件](plugins.md)
扩展服务器功能:
- 获取插件
- 安装和管理插件
- 配置插件
- 常用插件推荐

### 5. [连接服务器](connect.md)
让玩家加入你的服务器:
- 本地连接方法
- 局域网连接
- 公网连接配置
- 解除回环限制

### 6. [常见问题](troubleshooting.md)
遇到问题时的解决方案:
- 服务器无法启动
- 玩家无法连接
- 插件加载失败
- 性能优化建议

## 快速开始

如果你想快速体验,可以遵循以下最简步骤:

1. **安装 Java 17+** - [准备工作](preparation.md)
2. **下载 Nukkit-MOT** - [下载链接](https://motci.cn/job/Nukkit-MOT/job/master/lastSuccessfulBuild/artifact/target/Nukkit-MOT-SNAPSHOT.jar/)
3. **创建启动脚本** - 参考 [安装教程](installation.md)
4. **启动服务器** - 双击运行脚本
5. **连接游戏** - 使用 `127.0.0.1:19132`

## 预计用时

完成整个教程大约需要:
- **最快**: 15-30 分钟 (仅搭建基础服务器)
- **完整**: 1-2 小时 (包含配置和插件安装)
