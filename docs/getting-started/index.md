# Nukkit Server Setup Tutorial from Scratch

Welcome to the Nukkit server setup tutorial! This guide will walk you through setting up your own Minecraft Bedrock Edition server from scratch, step by step.

## Tutorial Overview

This tutorial is suitable for:
- Beginners with no server setup experience
- Developers who want to quickly get started with Nukkit
- Server administrators who need reference documentation

You will learn to:
- Set up a complete Nukkit server
- Configure server parameters
- Install and manage plugins
- Troubleshoot common issues

## Tutorial Chapters

Follow these chapters in order for a complete understanding of Nukkit server setup:

### 1. [Preparation](preparation.md)
Before starting, you need to prepare the necessary environment:
- Check system requirements
- Install Java runtime environment
- Verify successful installation

### 2. [Download and Installation](installation.md)
Obtain Nukkit and complete basic setup:
- Download Nukkit-MOT

- First server startup

### 3. [Configure Server](configuration.md)
Customize your server settings:
- Edit server.properties

### 4. [Install Plugins](plugins.md)
Extend server functionality:
- Get plugins
- Install and manage plugins
- Configure plugins
- Recommended common plugins

### 5. [Connect to Server](connect.md)
Let players join your server:
- Local connection method

### 6. [Troubleshooting](troubleshooting.md)
Solutions for common issues:
- Server won't start
- Players cannot connect

- Performance optimization tips

## Quick Start

If you want a quick experience, follow these minimal steps:

1. **Install Java 17+** - [Preparation](preparation.md)
2. **Download Nukkit-MOT** - [Download Link](https://motci.cn/job/Nukkit-MOT/job/master/lastSuccessfulBuild/artifact/target/Nukkit-MOT-SNAPSHOT.jar/)
3. **Create Startup Script** - Refer to [Installation Tutorial](installation.md)
4. **Start Server** - Double-click to run the script
5. **Connect to Game** - Use `127.0.0.1:19132`

## Estimated Time

Completing the entire tutorial will take approximately:
- **Fastest**: 15-30 minutes (basic server setup only)
- **Complete**: 1-2 hours (including configuration and plugin installation)
