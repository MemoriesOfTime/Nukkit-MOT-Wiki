# server.properties

Modify the running configuration of the server to customize some basic functionalities.

## `server-ip`
| Data Type | Default Value |
|-----------|---------------|
| String    | 0.0.0.0       |

Server IP address

## `enable-query`
| Data Type | Default Value |
|-----------|---------------|
| Boolean   | on            |

Whether to allow querying server information via Query

See: https://wiki.vg/Query

## `server-port`
| Data Type | Default Value |
|-----------|---------------|
| Integer   | 11013         |

Server port number

## `query.port`
| Data Type | Default Value |
|-----------|---------------|
| Integer   | 11013         |

Port for Query queries
